# Madlibs 

## Code

The app will take user input create a madlib with it.

### Sample game dialog

1. What do you want to tell a story about? (Choose from a list)
1. Great! Let's get started. Enter a verb. 
1. Enter a number
1. Enter a city. 
1. Last one. Enter a plural noun.
1. Here's your story!
